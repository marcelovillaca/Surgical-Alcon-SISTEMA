# Surgical Alcon SISTEMA

Sistema de gestão e inteligência para a Alcon Surgical Paraguai. O projeto visa centralizar operações de CRM, inventário, pedidos, visitas e análise de dados estratégicos.

## Visão Geral
O sistema é uma plataforma web moderna construída com React, Vite, Tailwind CSS e Supabase, projetada para fornecer aos gerentes e equipes de campo ferramentas robustas para monitoramento de metas, gestão de estoque e relacionamento com clientes.

## Stack Tecnológica
- **Frontend**: React + Vite + TypeScript
- **Estilização**: Tailwind CSS + shadcn/ui
- **Backend/Banco de Dados**: Supabase (PostgreSQL)
- **Gerenciamento de Estado/Dados**: TanStack Query (React Query)
- **Roteamento**: React Router DOM

## Módulos Principais
- **Dashboard Principal**: Visão geral de KPIs.
- **Inventário**: Controle de estoque e movimentações.
- **CRM & Visitas**: Gestão de clientes e roteiros de campo.
- **Pedidos**: Fluxo de solicitações e vendas.
- **Inteligência & Market Share**: Análise competitiva e de mercado.
- **Conofta**: Gestão específica para o mercado público/CONOFTA.
- **Auditoria**: Logs de atividades do sistema.

## Objetivos do GSD (Get Shit Done) neste projeto
1. **Manter o Contexto Limpo**: Evitar a degradação da qualidade com o crescimento do projeto.
2. **Desenvolvimento Orientado a Specs**: Garantir que cada nova feature seja bem definida antes da implementação.
3. **Histórico Transparente**: Commits claros e rastreabilidade de decisões.
